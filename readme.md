# First Install

$ gem install watchr
$ gem install compass
$ gem install haml

# Building while developing

$ compass watch
$ watchr watchr.rb

## Add a new project

$ ./new_project

## Build site
$ make

# Testing site locally
Point MAMP (or similar) to ./build/index.php
